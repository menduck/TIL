# 형 변환
## 자료형 변환
### 암시적 형 변환
  : 사용자가 의도하지 않고, 파이썬 내부적으로 자료형을 변환 하는 경우
```py
True + 3 # 4
3 + 5.0 # 8.0
3 + 4j + 5 # 8 + 4j
```

### 명시적 형 변환
  : 사용자가 특정 함수를 활용하여 의도적으로 자료형을 변환 하는 경우

```py
# 문자열은 암시적 타입 변환 X
'3' + 4 # TypeError

# 해결방안 => 명시적 타입 변환
int('3') + 4 # 7

# 정수 형식이 아닌 경우 타입 변환 X
int('3.5') + 5

# 해결방안 => 명시적 타입 변환
float('3.5') + 5 # 8.5
```
