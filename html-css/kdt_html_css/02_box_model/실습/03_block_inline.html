Block 요소

상에서 하로 쌓임

p태그도 대표적인 block태그

block요소는 기본으로 부모 요소 너비의 100%를 차지하고, 자식 콘텐츠의 최대 높이를 취함

block 요소의 총 너비/높이 = content(좌우/상하) + padding(좌우/상하) + border(좌우/상하)

block요소는 서로 margins로 구분

Inline 요소

좌에서 우로 쌓임

inline태그는 span태그, a태그, img태그

inline태그는 자기 콘텐츠의 너비와 높이만 차지함 그래서 inline요소는 width나 hight 속성을 지정할 수 없음
