한옥스테이 “화응(花應)”
화응(花應)은 전주의 한 가운데, 완산구 태평동에 위치하고 있습니다. 태평동은 과거 최초의 전주역과 연초제조창이 있던 교통과 상권의 중심이었으며 상인들의 마을인 주택가 골목에 그 당시 풍수와 지리를 활용하여 1969년 15평으로 건축되었습니다.
